BONE FURNITURE

Spencer Fung Architects 2023. Kensal rise, London, United Kingdom

Description:

Renders showing the various designs of furniture and lights all using nature to determine the form, specifically that of twigs and bones. Using existing branches to create unique forms for both lights and tables, finished in a dark bronze. These renders and 3d models have been created based on pencil sketches provided by boss; I was set to imagine these 2d drawings as 3d objects.
